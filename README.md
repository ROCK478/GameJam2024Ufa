# GameJam2024Ufa
**Задание на хакатон: Разработать за 1 сутки игру, на тематику "Сказки Пушкина"**

**Дата: Сентябрь 2024**

**Проект: Игра в жанре 2D-платформер под названием "КиберРуслан"**

**Технологии: Unity, C#**

**Результат: Победа**

## Инструкция пользователя
В папке Build находится файл .exe, при запуске которого начнется игра.

## Управление
- Движение WASD
- Прыжок Пробел
- Тихая ходьба Shift
- Удар ЛКМ
- Взломать демона E

## Сюжет
Руслан и Людмила играют свадьбу, Людмила пропадает, Руслан бежит за ней. Попадает в кибер-мир, в котором мечом сражается с демонами и пытается спасти Людмилу. 
На пути попадаются персонажи из сказок Пушкина (перерисованные под киберпанк стиль).

## Особенности 
- Возможность взломать демона путем решения задач по информатике (побитовые операции над двоичными числами)
- Персонажи и спрайты нарисованы художниками
- Использование нейросетей для генерации начального комикса
- Озвучка собственным голосом с последующим изменением в нейросети

## Завершение
Проект реализован за 24 часа, в дальнейшем не развивался. Присутствуют баги и незаконченность. По задумке в конце должна быть битва с главным боссом - Черномором.
Но это уже совсем другая история. Счастья всем и позитива!
